Welcome to the Workflow Team Web Tools

Get Started with one of the following tools:

Global Error View:
This view shows the errors occuring at sites in an attempt to visualize 4-dimensional data. From this page, you can look at more detailed reports of a given workflow step. From that reports page, you can submit actions to take on the workflow, if you have a registered account.

Create a New Account:
In order to submit actions, you must register a username. This registration must be verified by a valid email address (specified on the page). The new user will not be registered if the same email is used twice. You can still browse the website without a registered name.

Reset a Forgotten Password:
If you forgot your username or password, you can reset the password by email here. Your username will be in the email in the event you forgot that too.

Show Logs:
This view shows logs from the elastic search server.
